Metabolic Network

CNRS, Dyliss Team

    Development of a tool to reconstruct multiple metabolic networks with Pathway Tools using multiprocessing (mpwt).
    Development of metage2metabo, a pipeline to reconstruct metabolic networks from microbiotas genomes and identify keystone species among them. Use and give feedbacks for some python packages (padmet, menetools, miscoto, clyngor).
    Creation of a prototype applying the Metabolic Pathway Drift hypothesis (PathModel).
    Create container (Docker, Singularity) for these tools to use them on cluster.
