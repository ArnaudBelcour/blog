Tools

Python package

  • mpwt: Pathway-Tools multiprocessing package.
    A little package to launch multiple runs of Pathway-Tools on different genomes, using genbank files.

  • pathmodel.
    A tool to infer ab initio reaction using known reactions.

  • pbsea: Pandas-based Single Enrichment Analysis.
    A little package to compute enrichment analysis on pandas dataframe.

Syntax Highlight
